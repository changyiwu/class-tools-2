# ClassHub | 班級抽籤與多元工具箱

ClassHub 是一款專為教師與課堂教學設計的多功能前端工具箱，採用精緻的暗色玻璃擬態（Glassmorphism）美學，整合多種課堂常用工具，幫助教師提升課堂互動性與班級經營效率。

---

## 🎯 核心功能

1.  **🎯 幸運大抽籤**：支援幸運大轉盤、滾動拉霸機、神秘翻牌三種模式，並提供「不重複抽籤 (排除已中籤)」設定，記錄完整抽籤歷史。
2.  **🔊 噪音監測器**：啟用麥克風即時監測環境分貝，視覺化儀表呈現。可自訂「分貝閾值」與「延遲時間」，當超標時觸發全螢幕紅色警示，協助班級自主音量管理。
3.  **👥 隨機分組**：可依據「自訂組數」或「每組人數」將學生名單隨機拆分成不同小組。
4.  **🪑 隨機座位表**：自訂教室座位排數（Rows）與列數（Columns），支援一鍵隨機大風吹、個別格子設為空位等功能，並可直接列印座位表。
5.  **⏱️ 計時器與碼表**：提供倒數計時器（附有酷炫圓環進度條與常用時間快捷鍵）與高精準度的跑秒碼表（支援單圈計時）。
6.  **💯 團隊計分板**：自訂班級小組的分數累計，支援一鍵加分、扣分、新增小組與重置所有分數。
7.  **📁 名單與設定**：內建名單管理器，教師可自訂多個班級、修改班級名稱、以及匯入/編輯學生名單，且全域同步更新至各工具中。

## 🛠️ 開發與本機執行

本專案採用 **Vite** 作為開發伺服器與建置工具。

### 1. 複製專案並安裝依賴
```bash
# 複製儲存庫（依據您的路徑調整）
cd class-tools-2

# 安裝開發依賴
npm install
```

### 2. 本機開發與預覽
```bash
# 啟動 Vite 本機開發伺服器
npm run dev
```
啟動後在瀏覽器開啟 [http://localhost:5173/](http://localhost:5173/)。

### 3. 打包與生產環境預覽
```bash
# 打包專案（產生 dist/ 目錄）
npm run build

# 本機預覽打包後的成果
npm run preview
```
